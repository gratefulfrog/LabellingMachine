Demonstration of The Great Labelling Machine

Expand console window to see more messages

Simulation Controls: first click in simulator window to get focus

A/a: Accelerate the simulation.

B/b: Toggle Blocking messages.

D/d: Decelerate the simulation.

L/l: Toggle Label point callout messages.

P/p: Toggle Pausing at each message - any key or mouse to resume.

R/r: Toggle Ramp end or Contact point sitcker blocking.

S/s: Toggle Synch messages (this is removed from next verison onward).

T/t: Toggle Tag point callout messages.

Press any other key in sumulator window to pause/restart the simulation.
